# CONCEITOS EXTRAS



- **Untracked:** São arquivos que o GIT não tem ciência de que existem
- **Tracked:** São arquivos que o GIT tem ciência de que existem

- **Unmodified:** São arquivos que não foram modificados
- **Modified:** São arquivos que foram modificados
- **Staged:** É onde ficam os arquivos que estão se preparando para fazerem parte de outro tipo de agrupamento



*Comando de status*: **git status**

*Comando de mover*: **mv (exemplo: mv "arquivo" ./pasta/)**

*Adicionar arquivo INDEX*: **echo > README."extensão"**



**OBS:** Quando fizer uma mudança, assim que executar o comando **git status**, o git irá sugerir que você adicione os arquivos para área de **staged** e depois **commite**.
