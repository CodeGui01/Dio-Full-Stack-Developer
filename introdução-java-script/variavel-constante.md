# DIFERENÇA DE VARIÁVEL E CONSTANTE



- A variável sempre muda, enquanto que a constante tem um valor fixo, ou seja, não muda



*Exemplo:*          **const "PREÇO" = 12**

​                           **var "desconto" = 5**

​                           **var "total" = PREÇO - desconto**



- O preço não muda
- O desconto pode variar
- O total pode variar
