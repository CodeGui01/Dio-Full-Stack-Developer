# EXECUTANDO UM ARQUIVO .JS



### PASSO A PASSO

- Execute o comando: **console.log("exemplo");**

- Crie um novo **terminal**
- Abra o terminal com **GIT BASH**
- Navegue até a pasta onde está o **arquivo .js**
- Execute o comando: **node exemplo.js**
